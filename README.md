Liste de commandes en Pascal pour le Coleco

Voici un aperçu de la commande ADAM :

<h2>Liste des fichiers</h2>

Voici la liste des différents fichiers proposés dans Corail :

<table>
		<tr>
			<th>Nom</th>
			<th>Description</th>	
		</tr>
		<tr>
			<td><b>ADAM.PAS</b></td>
			<td>Cette commande permet de lancer l'interpréteur SmartBASIC du ADAM.</td>
		</tr>	
		<tr>
			<td><b>ASMZ80.PAS</b></td>
			<td>Cette commande permet de lancer le compilateur de langage de programmation assembleur pour les microprocesseur Z80.</td>
		</tr>    
	   <tr>
		       <td><b>RUNZ80.PAS</b></td>
		       <td>Cette commande permet d'exécuter du code machine du microprocesseur Z80.</td>
		</tr>    
</table>
	

<h2>Compilation</h2>
	
Les fichiers Pascal n'ont aucune dépendances, il suffit de télécharger le fichier désiré et de le compiler avec Free Pascal avec la syntaxe de commande  :

<pre><b>fpc</b> <i>LEFICHIER.PAS</i></pre>
	
Sinon, vous pouvez également le compiler avec le Turbo Pascal à l'aide de la syntaxe de commande suivante :	

<pre><b>tpc</b> <i>LEFICHIER.PAS</i></pre>
	
Par exemple, si vous voulez compiler ADAM.PAS, vous devrez tapez la commande suivante :

<pre><b>fpc</b> ADAM.PAS</pre>
